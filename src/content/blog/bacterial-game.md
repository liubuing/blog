> 细菌博弈赛道是合成生物学创新赛中最考验"系统思维"的赛道。它不是比单点最优,而是比多维权衡下的综合适应力。这篇文章记录了我们团队的设计思路。

## 赛道:三个维度的博弈

细菌博弈赛要求工程化的大肠杆菌在限定培养基(CM)中竞争,比拼三个互相制衡的表型:

1. **运动性(Chemotaxis)**:在半固体平板上通过趋化圈扩散"抢地盘"
2. **耐酸性(pH 4.5)**:在菌群代谢产酸的环境中存活
3. **底物利用率**:利用培养基中的阿拉伯糖等碳源

这三者存在**权衡(tradeoff)**:运动耗能,抗酸耗能,代谢产酸会伤己。你不可能三项都拉满——这就是博弈的本质。

## 策略:主动制造短板 + 理性补强 + 定向进化

我们的核心策略是三段式,在博弈论意义上是一种"以退为进":

### 第一段:主动 handicapping——敲除 acrA

`acrA` 是 AcrAB-TolC 多药外排泵的组件。敲除它,菌丧失物质外排能力,**耐酸性下降**。

这是有意制造的"可控弱点"——相当于在博弈中主动放弃一维,换取其他维度的改造空间和进化起点。

### 第二段:理性补强——gadE 组成型表达

在 `intS` 中性位点整合 `P_J23100-gadE`,让谷氨酸依赖型酸抗系统(GDAR)**组成型表达**。

核心洞察:**酸中和是比药物外排更有效的 CM 适应性策略**。

机制:gadE 是 GDAR 主调控因子,其组成型表达"绕过天然酸信号激活",让抗酸系统"随时待命"。GadB 消耗胞内 H⁺ 催化谷氨酸脱羧生成 GABA,GadC 反向转运谷氨酸进胞内、排出 GABA,维持胞内 pH 稳态。

为什么不选 AR3(精氨酸依赖)系统?因为余家瑞同学测算了 CM 培养基的底物浓度:酪蛋白水解物提供游离谷氨酸 0.54-0.65 mM(充足),而精氨酸仅 0.14-0.18 mM(不足)。GadB 最适 pH 3.8-4.5,恰好在产酸环境活性区间。决策有数据支撑。

### 第三段:UV 适应性进化

UV 诱变(15W/30cm, 254nm),3 代迭代,每代从 15 个候选中选"王"。

## 博弈论评分函数

每代选"王"的标准是一个对称加权的 log₂ 倍数变化评分:

```
Score = 0.4 × log₂(FC_motility) + 0.3 × log₂(FC_acid) + 0.3 × log₂(FC_util)

FC = 候选表型值 / ΔacrA 均值(以短板菌株为基准)
```

用 `log₂` 保证对称:翻倍 +1,减半 -1,避免极端值主导。权重体现博弈优先级:运动性 0.4 > 耐酸 0.3 = 利用 0.3。

以 `ΔacrA` 为基准(=1.0)是博弈论式的"相对优势"度量——不是比绝对值,而是比"相对短板提升了多少"。

## 方案迭代:13 项致命问题的修复

进化方案从 v2 到 v3 经历了严格迭代,逐条修复了 13 项问题(多条标"致命"):

| v2 问题 | v3 修复 |
| :- | :- |
| UV+EMS 联合致死率失控 | 改纯 UV(EMS 安全风险高+突变谱不互补) |
| 致死率相加谬误 | 认清 UV+EMS 顺序联合是相乘(90%×90%=99%致死,非 10%) |
| 无甘油冻存 | -80°C 甘油菌双备份 |
| 选择表型≠验证表型 | 15 候选 × 3 assay 交叉测定 |
| Davis/M9 母液混淆 | 严格区分 |
| 无测序 | Day12 送 WGS(Illumina PE150, ≥100×) |
| 时间表不可行 | 12 天 3 代(原 13 天 6 代) |

v3 最终菌株:`MG1655 ΔacrA::FRT intS::P_J23100-gadE::FRT`(clean deletion,无抗性标记,符合合成生物学"干净底盘"理念)。

## 代谢建模:FBA 通量平衡分析

我们用 **FBA(Flux Balance Analysis)** + COBRA Toolbox 做代谢网络建模,模拟 E. coli 在好氧/厌氧下的生长通量,分析"缺氧/产酸"博弈环境下的代谢瓶颈。

FBA 的原理:基于物理化学约束(区室化、质量守恒、热力学方向性)定义可行通量分布集,优化目标函数(如生物量)。这让我们能在实验前预判"gadE 过表达会抢走多少 TCA 通量"这类问题。

## 哲学小结

这个项目的设计哲学,恰好是"无性之美与有性之美"的绝佳隐喻:

**主动制造短板**(敲除 acrA)是"无性之美"——精确的、确定的、可复制的工程操作;**适应性进化**(UV 诱变 + 三维筛选)是"有性之美"——在随机变异中,让多维权衡的最优解**涌现**出来。

我们用工程的确定性铺设舞台(基因线路),用进化的不确定性寻找演员(最优表型组合)。两者相济,方为合成生物学。
